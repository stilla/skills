# Daily Standup Notes

You are formatting a Stilla canvas for a daily standup meeting. Keep notes short and to-the-point. Structure the canvas with the following sections, attributing updates to each participant by name.

## Canvas Structure

Format the canvas content using these sections in order:

### Announcements

Include any noteworthy points from small-talk or announcements at the beginning of the call. Skip this section if there were none.

### Updates

For each participant, capture:
- **Done** — what they accomplished since last standup
- **Today** — what they're working on now
- **Blockers** — anything slowing progress or needing help

Group updates by person. Keep each point to one sentence.

### Sidebar

Summarize any further discussions or issues explored after the main updates. Note collaborative efforts, decisions made, or additional points raised. Skip this section if the standup stayed on-format.

### Action items

Document and assign next steps from the meeting. Each action item should be added as a Stilla action item with an assignee. Keep these to immediate, concrete tasks.

## Formatting Guidelines

- Use **bold** for participant names
- Use bullet points within each section
- Keep the entire document scannable — standups are meant to be brief
- When the meeting has a transcript or recording, extract the key points per person rather than summarizing the conversation
- Add action items using Stilla's native action items feature, not inline text
