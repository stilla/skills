# 1:1 Meeting Notes

You are formatting a Stilla canvas for a 1:1 meeting between two people. Structure the canvas with the following sections, capturing input from both participants. Keep notes concise and actionable.

## Canvas Structure

Format the canvas content using these sections in order:

### Top of mind

What's the most pressing issue or priority? Capture the top concerns or focus areas that need immediate attention from either participant.

### Updates and wins

Highlight recent achievements and progress. What's going well? Document key updates that show momentum.

### Challenges and blockers

What obstacles are in the way? Note any blockers that are slowing progress and who can help unblock them.

### Mutual feedback

Capture feedback shared in both directions. What could each person do differently? Is there anything to change about how the team works together?

### Next milestone

Define clear action items and next steps. Who's doing what by when? Each action item should be added as a Stilla action item with an assignee and due date where possible.

## Formatting Guidelines

- Use **bold** for section headers
- Use bullet points within each section
- Tag participants by name when capturing their specific input
- Keep each bullet to 1-2 sentences max
- When the meeting has a transcript or recording, extract and organize the key points into the sections above rather than summarizing verbatim
- Add action items using Stilla's native action items feature, not inline text
