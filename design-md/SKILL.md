# Design MD — Stilla Brand

Complete design system reference for Stilla's visual language. Use when building, reviewing, or generating any UI that should match Stilla's brand — app components, marketing pages, landing pages, or product prototypes.

## How to use

1. Load `DESIGN.md` (in this same directory) for the full reference
2. Use Section 9 (Agent Prompt Guide) for quick copy-paste component specs
3. Check Section 7 (Do's and Don'ts) before finalizing any output
4. Always provide both light and dark mode variants

## When to load this skill

- Building any Stilla UI component or page (app or marketing)
- Reviewing UI code for brand consistency
- Generating Stilla-branded prototypes or mockups
- Any request involving Stilla's visual identity

Always reference `DESIGN.md` — don't guess at tokens, colors, or typography.
